# Customer Personality Analysis

## About the Project
This project is about analyzing customer data using machine learning. The goal is to understand customer behavior, predict spending, classify responses, and group customers into segments.

---------------------------------------------------

## What I did in this project

- Cleaned the data and removed wrong values
- Handled missing values in Income
- Created new features like Age, TotalSpending, and TotalChildren
- Did Exploratory Data Analysis (EDA) using plots
- Built regression models to predict TotalSpending
- Built classification models to predict customer response
- Applied K-Means clustering to segment customers

---

## Models Used

### Regression
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

Best model: Decision Tree

---

### Classification
- Logistic Regression
- KNN
- Random Forest

Best model: Random Forest (accuracy), but recall was also considered

---

### Clustering
- K-Means with k = 3
- Customers were grouped into 3 segments based on behavior

---

## Main Insight
Customers can be divided into different groups based on income and spending, and this helps in understanding their behavior better.
